# Gym Webapp (V2)

Mobile-first Next.js (App Router) + Tailwind starter:
- Home page with 2-column tile layout
- Bottom nav
- Dark/light mode toggle (no hydration mismatch)

## Local run

```bash
npm i
npm run dev
```

## Deploy to Vercel

Import the project on Vercel (or push to Git and connect), then deploy.
